# Crypto Price Calculator

A Python script that fetches cryptocurrency prices from the CoinCap API based on ticker symbol and calculates the total USD value of a specified amount of coins.

## Features

- Fetches live cryptocurrency prices
- Supports any coin available through the CoinCap API
- Handles invalid input and network errors

## Requirements

- Python 3
- requests

Install dependencies:

```
pip install requests
```

## Setup

1. Create an account and get an API key from CoinCap.
2. Replace:

```
ENTER_API_KEY_HERE
```

with your API key.

## Usage

Run:
```
python crypto.py
```
